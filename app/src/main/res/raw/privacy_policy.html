Android Permissions

WAKE_LOCK

This permission allows an application to hold a wake lock to keep the device from sleeping.

This permission allows an application to run in the background as a foreground service.

This permission allows the app to notify the user when a timeout is started. It is optional and up to the user whether they want to be notified or not.

This permission allows an application to request that the system ignore battery optimizations.
